# 80sChorusMachine 🎸✨

Un plugin audio ispirato alle sonorità classiche degli anni '80. Sviluppato in C++ utilizzando il framework **JUCE**. 

## Caratteristiche
- **Classic 80s Vibe**: Algoritmo di modulazione progettato per emulare i chorus analogici vintage.
- **Parametri Controllabili**: Gestione di Rate, Depth e Mix.
- **Cross-Platform**: compilabile in formato VST3 e AU.

## Struttura del Progetto
- `80sChorusMachine.jucer`: File di configurazione del Projucer per rigenerare i progetti Xcode/Visual Studio.

## Requisiti per la Compilazione
Per compilare il plugin sul tuo computer
1. **JUCE Framework** (scaricabile da juce.com).
2. **Projucer** (incluso in JUCE).

### Istruzioni
1. Scarica o clona il repository.
2. Apri il file `80sChorusMachine.jucer` con il Projucer.
3. Se necessario, imposta i percorsi dei moduli JUCE nelle impostazioni globali.
4. Clicca su **"Save and Open in IDE"** per generare il progetto Xcode/VS.
5. Compila il target `VST3` o `AU`.
